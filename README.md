## 📦 О проекте

**DIM & DAN HOMES** — это терминальная CRM-система для управления клиентами, заказами и бонусной программой.  
Разработана для автоматизации работы строительной компании в мире Minecraft.

> ⚡ Вся логика построена вокруг **лояльности клиентов**, **реферальной программы** и **автоматических бонусов**.

---

## 🧠 Возможности

| Модуль | Описание |
|--------|----------|
| 👥 Клиенты | Регистрация, редактирование, список активных клиентов |
| 💰 Бонусы | Начисление, списание, история, ежедневный бонус (+50 ед.) |
| 🎁 Промокоды | 10+ активных акций (Welcome, BlackFriday, Birthday и др.) |
| 🔗 Рефералы | Код приглашения, +300 за друга, 5% от его покупок |
| 📈 Статистика | Выручка, средний чек, топ-10 пригласивших, VIP/Silver статусы |
| 💾 Данные | Автобэкапы, экспорт в CSV, восстановление из файлов |
| 🖥️ UI | Rich + Colorama → градиенты, таблицы, интерактивные меню |
